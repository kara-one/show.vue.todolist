# Vue TodoList
- Simple todo-list using localStorage to save your todos
- vue, vuex, vue-router, materialize-css, sass
- app work at Heroku [DEMO](https://show-vue-todolist-01.herokuapp.com/)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development

npm run serve
